使用hexo和github构建自己的博客

2017-12-14
Hexo

Hexo中文文档

Install Hexo

  1. Install Git

  2. Install Node.js

  3. Install Hexo

Init Blog Project

Config Github

  1. create a new repository,set repository name to “<githubname>.github.io” strictly.

  2. generate ssh key

  3. Enter the settings of repository,copy ssh key which just be generated to “deploy keys” in order to ensure you deploy blog to github directly.

  4. To verify whether adding key is success.

Config Blog Project

  1. modify _config.yml

  2. install deploy tool

Local Preview

open browser and enter: http://localhost:4000

Deploy to github

open browser and enter: http://<githubname>.github.io
